## Getting Started

### Prerequisites

- Node.js 18+
pnpm

### Installation
```bash
# Install dependencies
npm install

# Copy environment variables
cp .env.example .env

# Update .env with your API URLs
```

### Development
```bash
npm run dev
```

App will be available at `http://localhost:3001`

### Build
```bash
pnpm run build
```

### Preview Production Build
```bash
pnpm run preview
```
